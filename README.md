# Практична робота №3 - 6

## Опис проєкту
У цьому проєкті створено простий мінісайт, який складається з трьох HTML-сторінок зі спільною навігацією.

Структура проєкту:
- **index.html** — головна сторінка сайту з текстовим контентом, списками, зображенням та таблицею.
- **pages/about.html** — сторінка "Про мене", яка містить текст, відео та аудіо.
- **pages/contact.html** — сторінка "Контакти" з формою зворотного зв’язку.

У проєкті реалізовано:
- коректну структуру **HTML5**;
- семантичні елементи (**header, nav, main, footer**);
- навігацію між сторінками;
- текстовий контент із заголовками та абзацами;
- форматування тексту (**strong, em**);
- маркований та нумерований списки;
- зображення з **alt**, **figure** та **figcaption**;
- таблицю з використанням **caption, thead, tbody**;
- форму з HTML5-валідацією (required, minlength, maxlength);
- медіаконтент (**video** та **audio**).

Також використано папку **assets** для зберігання зображень та медіафайлів.

## Стилі
### Загальні стилі
- Встановлено `box-sizing: border-box` для всіх елементів.
- Основний шрифт: Arial, кольори та змінні через CSS-перемінні (`--color-primary`, `--color-accent`, `--color-bg`, тощо).
- Контейнер `.container` з обмеженою шириною, центрованим контентом та адаптивними паддінгами.
- Адаптивність через медіа-запити для ширини екранів 768px та 1024px.

### Header
- Горизонтальне меню з пунктами `.header__nav-item` та посиланнями `.header__nav-link`.
- Стани `normal`, `hover`, `active`.
- Відступи між пунктами, заокруглення, зміна кольору при наведенні.

### Footer
- Простий футер з центруванням тексту та рамкою зверху.

### Home (index.html)
- Головний заголовок `.home__title` та підзаголовки `.home__subtitle`.
- Текстові блоки `.home__text` та кнопки `.home__btn`.
- Контентні картки `.home__content-card` з тінями та заокругленнями.
- Списки `.home__list` з відступами та міжрядковими інтервалами.
- Зображення `.home__figure img` з адаптивною шириною, заокругленням та тінню.
- Таблиця `.home__table` з підписом, рамками, чергуванням кольорів рядків та контрастним заголовком.
- Дві колонки контенту `.home__content` з Flexbox та wrap.

### About (about.html)
- Блок `.about` з контентною секцією та медіакартками.
- Текстові параграфи `.about__paragraph`, акценти `.about__strong` та `.about__emphasis`.
- Flexbox для `.about__content` (текст + медіа).
- Медіа-картки `.about__card` для відео `.about__video` та аудіо `.about__audio` з підписами `.about__caption`.
- Адаптивна ширина медіа та обмеження max-width.

### Contact (contact.html)
- Форма `.contact__form` у окремій картці з фоном, тінню та заокругленнями.
- Поля `.contact__input`, `.contact__select`, `.contact__textarea` з однаковою шириною, внутрішніми відступами, рамкою, стилями при `:focus`, `:valid` та `:invalid`.
- Лейбли `.contact__label`, чекбокси `.contact__checkbox`, радіо `.contact__radio`.
- Кнопки `.contact__button` з hover-ефектом.
- Flexbox для кнопок `.contact__actions`.
- Адаптивна ширина форми, максимальна ширина та медіа-запити для планшетів і мобільних.

### Адаптивність
- Media queries для 768px і 1024px:
  - Вертикальна побудова меню у мобільній версії.
  - Зменшені відступи в контейнерах та картках.
  - Гнучка ширина відео, зображень та таблиць.
  - Коригування font-size для заголовків на вузьких екранах.

## Як запустити проєкт
1. Відкрити папку проєкту у **Visual Studio Code**.
2. Встановити розширення **Live Server** (якщо воно ще не встановлене).
3. Натиснути правою кнопкою миші на файл **index.html**.
4. Обрати **Open with Live Server**.

Після цього сайт відкриється у браузері за локальною адресою.
